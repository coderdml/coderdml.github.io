axios发送并发请求
使用axios.all多个请求的数组

axios返回的结果是一个数组，可以用axios.spread将[res1,res2]展开为res1,res2

<script>
    import axios from 'axios'

    export default {
        name : 'app',
    }

    axios.all([axios({
        url:''
    }),axios({
        url:'',
        params:{
            type: ' ',
            page: 
        }
    })])
    <!-- .then(results => {
        console.log(results);
        console.log(results[0]);
        console.log(results[1]);
    }) -->
    .then(axios.spread((res1,res2)=>{
        console.log(res1);
    }))

axios配置选项
method默认为get
当method为get时，对应使用URL查询对象params
当method为post时，对应使用data
